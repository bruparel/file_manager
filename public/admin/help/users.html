1. A user is someone who has a valid login created by you (the admin). It is very important to assign a correct role to the user while creating or editing the user record.

2. A unique username and a unique email should be chosen so that he can be correctly identified. If the user wants to change username/email at a later date, the admin can do it for him including changing/resetting his password which is commonly forgotten.

3. The user can reset the password at any time by clicking on Change Password link provided he remembers his current password. If he has forgotten it then the admin has to reset it for him. Change Status link is used mark user as inactive so the system will not actively track him.

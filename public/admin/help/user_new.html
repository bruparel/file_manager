1. There are four required fields for a successful user account creation: username, email address, and password_confirmation.

2. The username and email address have to be unique. If you happen to choose a username that has already been assigned to someone else in the system then you will get a message accordingly, e.g., Username has already been taken. The same thing applies to email address as well. If you are entering a valid and unique email address and the system responds with the message that the Email Address has aready been taken then you should contact the Administrator.

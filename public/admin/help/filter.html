1. It is worth pointing out that the Search functionality works the same way across the Clients, Documents, and Users(admin only) tabs.

2. Make sure that the Filter button is checked. If it is not then click on it.

3. Select a value for the filter from the drop-down list. For example, select Uploaded.

4. Now click on the Submit button to filter (or select) only those documents that have an Uploaded status.

5. This allows you to work on a sub-set of documents that have a specific status. You assign the status in Edit screens for Documents.

6. If you find that you need additional document statuses then please ask the admin to add those for you.

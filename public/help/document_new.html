1. By far the most important thing to be aware of is that all fields that are shown with bold labels are required. Therefore, if you leave anyone of them blank inadvarently then the system will print an appropriate error message in red next to the field label.

2. You upload a new document by clicking on the new link, entering a title, and then by clicking on the Browse... button to locate the file. You can replace an existing file by a new file by clicking on the edit link and reassigning a new file. You delete a file by clicking on the delete button.

3. To re-attach a document which may have been edited and you want a more current copy to be uploaded, or you want to replace the current document with a different document, click on the Browse button which will display your disk(s) directory structure. Navigate to the desired document and select it. It will then be linked to this document record and uploaded to the server replacing your previous document.

4. The documents are attached to a particular client (user really).

5. The document status defines the work-flow and make sure to select the proper value. If you find that a particular value of document status is missing and should be available then ask the admin user to create it for you.

6. The document type, shown on the screen as folder, lets you assign a document to a particular folder for a client. If you find that a particular value of document status is missing and should be available then ask the admin user to create it for you.

7. It is advisable to enter descriptive information pertaining to this document for later reference. Note that the title field is required.

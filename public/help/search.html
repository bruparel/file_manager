1. It is worth pointing out that the Search functionality works the same way across the Clients, Documents, and Users(admin only) tabs.

2. Make sure that the Search button is checked. If it is not then click on it.

3. Select the desired search by attribute from the drop-down, e.g., client name.

4. Type in the client name and either press enter or click on the 'Submit' button.

5. Select another attribute, e.g., city to search by city and repeat above steps.

6. The search string does not have to be entered exactly. For example, when searching by email in the User tab (admins only), if you enter 'yahoo' then it will find all users who have a yahoo email address, e.g., bcruparel@yahoo.com.
